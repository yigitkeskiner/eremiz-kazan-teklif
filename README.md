# Erem Group Vercel Site

Bu repo statik bir web sitesi olarak hazırlandı. Vercel'e doğrudan proje olarak bağlanabilir.

## Dosyalar

- `index.html`: Sayfa içeriği
- `styles.css`: Profesyonel kurumsal arayüz tasarımı
- `script.js`: Teklif hesaplama mantığı
- `vercel.json`: Vercel için basit yönlendirme ayarı

## Kullanılan PDF Verileri

- Kurumsal renkler: `#137D76` ve `#00BABC`
- Kuruluş hikayesi ve büyüme adımları
- 50+ marka ve 2500+ ürün vurgusu
- İstanbul, Gebze, İzmir ve Antalya yapılanması
- Ataşehir merkez iletişim bilgileri
